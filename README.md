# CV Corporativo - Valentina

## 📋 Descripción
CV corporativo profesional para Valentina - Empresa de colaciones empresariales.

## 🚀 Cómo usar

### Opción 1: Abrir directamente
1. Haz doble clic en el archivo `index.html`
2. Se abrirá en tu navegador predeterminado

### Opción 2: Usar un servidor local
Si quieres hacer pruebas más avanzadas, puedes usar un servidor local:

```bash
# Si tienes Python instalado:
python -m http.server 8000

# Si tienes Node.js instalado:
npx serve
```

## 🎨 Personalización

### Cambiar colores
Busca en el código las siguientes líneas para cambiar los colores principales:
- `#667eea` y `#764ba2` son los colores del gradiente principal

### Agregar logo
Reemplaza la letra "V" en el círculo del logo con tu imagen:
1. Guarda tu logo como `logo.png`
2. Reemplaza esta línea en el HTML:
```html
<div class="logo">V</div>
```
por:
```html
<div class="logo"><img src="logo.png" alt="Logo Valentina" style="width: 100%; height: 100%; border-radius: 50%;"></div>
```
